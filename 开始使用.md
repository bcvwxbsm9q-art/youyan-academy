# 🚀 游雁学院 - 快速开始指南

## ⚡ 3 分钟快速上线

### 第 1 步：安装依赖（1 分钟）

打开终端（命令提示符），输入：

```bash
cd "E:\培训相关\桌面\learning"
npm install --registry=https://registry.npmmirror.com
```

等待安装完成...

### 第 2 步：初始化数据库（30 秒）

```bash
node scripts/init-database.js
```

看到 `🎉 数据库初始化完成！` 表示成功。

### 第 3 步：启动服务器（10 秒）

```bash
node server-production.js
```

看到以下信息表示成功启动：

```
============================================================
🚀 游雁学院服务器已启动
============================================================
📡 服务地址：http://localhost:3003
🔐 登录页面：http://localhost:3003/login.html
============================================================
```

### 第 4 步：访问平台

打开浏览器，访问：**http://localhost:3003/login.html**

**使用默认账户登录：**
- 管理员：`admin` / `admin123`
- 学生：`student` / `student123`

## 🔧 常用操作

### 修改端口号

编辑 `server-production.js`，找到：

```javascript
const port = process.env.PORT || 3003;
```

改为其他端口，如 `3004`。

### 查看健康状态

访问：http://localhost:3003/api/health

返回示例：

```json
{
  "status": "ok",
  "database": "connected",
  "stats": {
    "users": 2,
    "courses": 0,
    "question_banks": 0,
    "questions": 0
  }
}
```

### 上传文件

使用 `/api/upload` 接口：

```javascript
const formData = new FormData();
formData.append('file', fileInput.files[0]);

const response = await fetch('/api/upload', {
    method: 'POST',
    body: formData
});

const result = await response.json();
console.log(result.url); // 文件访问 URL
```

## 📊 API 接口速查

### 认证接口

```
POST /api/auth/register      - 用户注册
POST /api/auth/login         - 用户登录
POST /api/auth/logout        - 用户登出
GET  /api/auth/me            - 获取当前用户
PUT  /api/auth/profile       - 更新资料
PUT  /api/auth/password      - 修改密码
```

### 题库接口

```
GET    /api/questions/banks              - 获取所有题库
POST   /api/questions/banks              - 创建题库
GET    /api/questions/banks/:id          - 获取题库详情
PUT    /api/questions/banks/:id          - 更新题库
DELETE /api/questions/banks/:id          - 删除题库
GET    /api/questions/banks/:id/questions - 获取题目列表
POST   /api/questions                    - 创建题目
```

### 通用接口

```
GET  /api/data              - 获取所有数据
GET  /api/data/:key         - 获取指定类型数据
POST /api/upload            - 上传文件
GET  /api/health            - 健康检查
```

## 🐛 常见问题

### Q: npm 命令不存在？

**A:** 需要先安装 Node.js，下载地址：https://nodejs.org/

### Q: 数据库初始化失败？

**A:** 检查是否已安装依赖：
```bash
npm install
```

### Q: 端口被占用？

**A:** 修改端口号或关闭占用程序：
```bash
netstat -ano | findstr :3003
taskkill /PID 进程 ID /F
```

### Q: 登录后跳转失败？

**A:** 检查浏览器控制台是否有错误，确保服务器正在运行。

### Q: 如何退出登录？

**A:** 清除浏览器缓存或调用 `/api/auth/logout` 接口。
