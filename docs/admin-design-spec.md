# 酷学院风格管理后台设计方案

## 设计理念
- 左侧侧边栏分组清晰，每组包含多个功能菜单
- 顶部固定导航，简化主导航项
- 卡片式布局展示数据
- 统一的操作按钮和状态标签

## 模块设计

### 1. 知识库模块 (admin-knowledge.html)
**侧边栏结构：**
- 内容管理
  - 知识文档
  - 分类管理
  - 标签管理
- 课程管理
  - 课程列表
  - 课程分类
  - 审核管理
- 系统设置
  - 基本设置
  - 安全设置

### 2. 培训模块 (admin-training.html)
**侧边栏结构：**
- 培训管理
  - 培训项目
  - 项目分类
  - 报名管理
- 考试管理
  - 题库管理
  - 试卷管理
  - 考试记录
- 调研管理
  - 问卷列表
  - 答题记录
- 证书管理
  - 证书模板
  - 发放记录

### 3. 师资模块 (admin-teacher.html)
**侧边栏结构：**

  - 讲师排行
  - 授课统计

### 4. 运营模块 (admin-operation.html)

  - 消息推送

### 5. 报表模块 (admin-report.html)
**侧边栏结构：**

  - 数据总览

## 核心功能
1. CRUD操作：课程、培训项目、考试、调研、讲师的新建/编辑/删除
2. 数据联动：前后端数据同步更新
3. 报表导出：Excel/CSV格式导出
4. 批量操作：批量启用/禁用、批量删除
5. 筛选搜索：多条件组合筛选
6. 分页管理：支持分页展示

## 技术实现
- 前端：Tailwind CSS + FontAwesome
- 数据存储：localStorage (模拟后端API)
- 图表：Chart.js
- 导出：前端导出为Excel格式
